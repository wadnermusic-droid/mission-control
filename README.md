## Troubleshooting

**"prisma: command not found"**
→ Run `npm install` first

**"Port 3000 in use"**
→ Use `npm run dev -- -p 3001`

**"Database errors"**
→ Delete `prisma/dev.db` and re-run `npm run db:migrate && npm run db:seed`
